# GmailGenerator

**GmailGenerator** es una herramienta escrita en **Python** que permite generar direcciones de correo **@gmail.com** aleatorias, válidas y configurables, de forma rápida y estética desde la consola.

El proyecto utiliza un archivo de configuración `society.ini` y cuenta con validaciones completas para una experiencia de usuario limpia y profesional.

---

## Características

- Generación de correos **@gmail.com**
- Nombres aleatorios de longitud configurable
- Caracteres válidos para Gmail
- Interfaz estética en consola (colores + banner)
- Configuración mediante `society.ini`
- Validación de entrada (números, límites, errores)
- Límite máximo de generación (1–100)
- Opción de generar múltiples veces sin cerrar el programa
- Guardado automático en archivo `.txt`
